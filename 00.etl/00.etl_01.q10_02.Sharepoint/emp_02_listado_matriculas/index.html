    Configuración del Entorno para la Automatización del Proceso de Scraping¶

    Este bloque de código configura el entorno necesario para llevar a cabo un proceso de scraping automatizado utilizando herramientas modulares y un sistema de logging bien estructurado.

    Primero, se importan los módulos básicos de Python como logging, os, sys, glob, re, datetime, pandas y openpyxl, esenciales para la configuración del entorno. También se agrega al sys.path la ruta de una carpeta superior que contiene el módulo Utils.Funciones. Esto permite organizar el proyecto en módulos reutilizables y facilita la importación de funciones específicas. Entre las funciones importadas destacan log_step_decorator download_all_files_from_sharepoint y limpiar_carpeta.El sistema de logging se configura para registrar eventos clave del proceso. Un logger identificado por el nombre del módulo (__name__) se establece con un nivel de detalle INFO, suficiente para capturar información relevante y errores críticos. Los mensajes se registran en dos destinos: un archivo llamado scraper.log para el seguimiento a largo plazo y la consola para monitoreo en tiempo real. Ambos manejadores comparten un formato estandarizado que incluye la hora, el nivel de registro y el mensaje, lo que facilita la depuración y el análisis.

    Este código asegura que el entorno esté completamente preparado antes de ejecutar el proceso principal de scraping, proporcionando trazabilidad, modularidad y flexibilidad para manejar diferentes flujos de trabajo automatizados.

    Automatización¶

    El código realiza un proceso de ETL (Extracción, Transformación y Carga) para los datos de matrículas almacenados en SharePoint. Utiliza decoradores para registrar los pasos del proceso, lo que facilita el seguimiento y la depuración. La función obtener_sharepoint_listado_matriculas descarga los archivos de Excel desde una carpeta específica en SharePoint, identifica los archivos que cumplen con el patrón 'Listado_Matriculas_*.xlsx', y selecciona el archivo más reciente basado en la fecha incluida en el nombre del archivo.

    Primero, se asegura de que la carpeta de destino exista y esté limpia. Luego, descarga los archivos desde SharePoint y filtra el archivo más reciente utilizando una expresión regular para extraer la fecha del nombre del archivo. Los archivos más antiguos se eliminan para mantener la carpeta organizada.

    El archivo más reciente se lee en un DataFrame de pandas, y se registran las columnas del archivo para su seguimiento. La función limpiar_columnas se encarga de eliminar y renombrar columnas específicas para limpiar y estandarizar los datos. Además, se eliminan los puntos de la columna TIPO_DOCUMENTO, se dividen los valores de la columna SEDE - JORNADA - PROGRAMA en tres columnas separadas (SEDE, JORNADA y PROGRAMA), y se eliminan duplicados basados en las columnas FECHA_MATRICULA y DOCUMENTO_ESTUDIANTE.

    Finalmente, si la variable de control ejecutar está activada, se llama a la función actualizar_sharepoint_procesado para actualizar los datos procesados de vuelta en SharePoint, asegurando que los datos de matrículas estén limpios y estructurados adecuadamente para su posterior análisis o carga en un sistema de almacenamiento de datos.

    def procesar_listado_matriculas(file_path):
    wb = openpyxl.load_workbook(file_path, data_only=True)
    all_data = []

    # Iterar sobre las hojas del archivo con índice
    for sheet_idx, sheet_name in enumerate(wb.sheetnames):
        sheet = wb[sheet_name]
        sheet_data = []
        header_row = None
        registro_num = 0  # Contador para el número de registro
        empty_row_count = 0  # Contador de filas vacías consecutivas

        for row_idx, row in enumerate(sheet.iter_rows(values_only=True), start=1):
            if row[0] and row[0].strip() == "Estudiante":  # Detectar encabezado
                header_row = row
                _info = sheet.cell(row=row_idx - 3, column=1).value
                if _info:
                    header_info = _info
                    registro_num = 0

                # Analizar header_row y obtener los nombres de las columnas
                if header_row:
                    col_indices = {
                        col_name.replace("\n", " "): idx
                        for idx, col_name in enumerate(header_row)
                        if col_name and col_name.replace("\n", " ") in [
                            "Estudiante", "Identificación", "Fecha Matrícula",
                            "Teléfono", "Celular", "Correo", "Nivel", "Estado"
                        ]
                    }
                continue

            if header_row:
                registro_num += 1                
                # Validar si la fila tiene suficientes columnas
                if row[0] is None:
                    empty_row_count += 1
                    if empty_row_count >= 5:  
                        break
                    continue
                elif row[0] == 'www.q10.com':
                    break
                else:
                    empty_row_count = 0  # Reiniciar contador si se encuentra una fila válida

                # Procesar datos

                # Dividir la columna "Identificación" en tipo y número
                identificacion = row[col_indices.get("Identificación", 1)]
                tipo, numero = (None, None)  # Valores predeterminados en caso de error
                if identificacion and " " in identificacion:
                    tipo, numero = identificacion.split(" ", 1)

                data_row = {
                    "Hoja": sheet_name,
                    "Registro": registro_num,
                    "Sede - jornada - Programa": header_info,
                    "Estudiante": row[col_indices.get("Estudiante", 0)],
                    "Tipo Identificación": tipo,
                    "Identificación": numero,
                    "Fecha Matrícula": row[col_indices.get("Fecha Matrícula", 2)],
                    "Teléfono": row[col_indices.get("Teléfono", 3)],
                    "Celular": row[col_indices.get("Celular", 4)],
                    "Correo": row[col_indices.get("Correo", 5)],
                    "Nivel": row[col_indices.get("Nivel", 6)],
                    "Estado": row[col_indices.get("Estado", 7)],
                }
                sheet_data.append(data_row)

        # Combinar los datos de cada hoja
        if sheet_data:
            all_data.extend(sheet_data)

    # Crear un DataFrame final con todos los datos procesados
    df = pd.DataFrame(all_data)
    df = df.dropna(subset=["Identificación"])  # Eliminar filas con identificación nula
    return df

@log_step_decorator("obtener_sharepoint_listado_matriculas")
def obtener_sharepoint_listado_matriculas():
    folder_url = etl_to_folder_url.get('emp_Listado_Matriculas', "URL por defecto si no se encuentra el valor de etl")
    # obtener_sharepoint(folder_url,_folder)

    # Obtener todos los archivos en la carpeta
    files = glob.glob(os.path.join(_folder, "Listado_Matriculas_*.xlsx"))

    # Verificar si la lista de archivos está vacía
    if not files:
        logger.error("No se encontraron archivos con el patrón 'Listado_Matriculas_*.xlsx' en la carpeta.")
        return None

    # Procesar todos los archivos y acumular los datos en un DataFrame
    all_dfs = []
    for file in files:
        df_temp = log_step_decorator(f"Procesando {file}")(procesar_listado_matriculas)(file)
        all_dfs.append(df_temp)

    # Concatenar todos los DataFrames en uno solo
    df = pd.concat(all_dfs, ignore_index=True)

    # Define las columnas a eliminar y los nuevos nombres de las columnas.
    columnas_a_eliminar = ['Hoja','Registro']
    #    'Sede - jornada - Programa',
    renombres = {
        'Estudiante': 'NOMBRE_ESTUDIANTE',
        'Tipo Identificación':'TIPO_DOCUMENTO',
        'Identificación':'DOCUMENTO_ESTUDIANTE',
        'Fecha Matrícula':'FECHA_MATRICULA',
        'Teléfono':'TELEFONO',
        'Celular':'CELULAR',
        'Correo':'CORREO',
        'Nivel':'SEMESTRE',
        }

    # Limpia las columnas del DataFrame, eliminando y renombrando según sea necesario.
    df = limpiar_columnas(df, columnas_a_eliminar, renombres, True)
    # Variable cargada "df" del estado del Kernel

    # Remove dots from TIPO_DOCUMENTO
    df['TIPO_DOCUMENTO'] = df['TIPO_DOCUMENTO'].str.replace('.', '', regex=False)

    # Split column into SEDE, JORNADA, and PROGRAMA
    df[['SEDE', 'JORNADA', 'PROGRAMA']] = df['SEDE - JORNADA - PROGRAMA'].str.extract(r'(\w+ \w+) (\w+) - (.+)')

    # Eliminar columna: 'SEDE - JORNADA - PROGRAMA'
    df = df.drop(columns=['SEDE - JORNADA - PROGRAMA'])

    #elimina de df los duplicados por las columnas 'Fecha Matrícula' y 'Identificación'
    df = df.drop_duplicates(subset=['FECHA_MATRICULA', 'DOCUMENTO_ESTUDIANTE'])
    return df

@log_step_decorator("ETL")
