    c4c_from_sharepoint

    Este script automatiza la descarga de archivos desde una carpeta de SharePoint utilizando el API de SharePoint y los módulos requests y msal para autenticar y obtener datos. El script también incluye funcionalidades de procesamiento de archivos, creación de carpetas y manejo de archivos descargados. A continuación, se describe el funcionamiento y propósito de cada componente y función del script.

    Dependencias y configuración¶

    1. Librerías:

      • El script importa varias librerías como argparse, io, logging, os, requests, pandas, msal, entre otras, para realizar tareas de autenticación, procesamiento de datos y manejo de archivos.

    2. Archivo credenciales.env:

      • Se cargan las credenciales de acceso desde el archivo credenciales.env, que contiene las claves necesarias para autenticar la aplicación en Microsoft Azure.

    3. Autenticación en Microsoft Azure:

      • Utiliza msal para obtener un token de acceso que se usará en las solicitudes de la API de SharePoint.

    4. Configuración de logging:

      • Se configura un logger para registrar los eventos en un archivo scraper_SSIS.log y en la consola. Esto permite el monitoreo del proceso de descarga y cualquier error que pueda ocurrir.

    Funciones principales¶

    1. download_file_from_sharepoint:

      • Esta función se encarga de descargar un archivo desde SharePoint utilizando la URL del archivo y el token de autenticación.
      • La función recibe el nombre del archivo, la URL de la carpeta de SharePoint y la ruta de destino para guardar el archivo descargado.
      • Si la solicitud es exitosa (status_code == 200), guarda el archivo en el directorio indicado.

    2. download_all_files_from_sharepoint:

      • Descarga todos los archivos de una carpeta específica en SharePoint.
      • Llama a la API de SharePoint para obtener la lista de archivos en la carpeta especificada.
      • Filtra los archivos que coinciden con un modelo de archivo (por ejemplo, Listadesolicitudesdeservicio__ES.xlsx) y descarga cada uno de ellos utilizando la función download_file_from_sharepoint.

    3. run_etl:

      • Esta función envuelve el proceso de descarga de archivos de SharePoint, llamando a download_all_files_from_sharepoint con la URL de la carpeta y el directorio de destino.

    4. Autenticación con msal:

      • El script usa la librería msal para obtener un token de acceso mediante las credenciales proporcionadas en el archivo credenciales.env. Este token se usa para autorizar las solicitudes a la API de SharePoint.

    5. Ejecución del proceso de descarga:

      • Dentro de la función run_etl, se configuran los parámetros de carpeta y directorio de destino, y se inicia el proceso de descarga de los archivos de SharePoint.

    Flujo del script¶

    1. Autenticación:

      • Se obtienen las credenciales y el token de acceso a la API de SharePoint utilizando msal con las credenciales proporcionadas en el archivo credenciales.env.

    2. Obtención y descarga de archivos:

      • El script obtiene los archivos de SharePoint desde la carpeta especificada (folder_url), filtra los archivos relevantes y los descarga al directorio de destino.

    3. Proceso ETL:

      • El proceso ETL se lleva a cabo con el decorador log_step_decorator, lo que permite registrar el avance de cada paso en el proceso de descarga.

    Consideraciones¶

    • Dependencias: Es necesario tener las librerías msal, pandas, requests, y xlwings instaladas para que el script funcione correctamente.
    • Carpetas de destino: El script crea el directorio de descarga si no existe previamente, asegurando que los archivos descargados se almacenen en la ubicación correcta.
    • Manejo de errores: Se incluye un manejo básico de errores a través del registro en logs para que cualquier error durante la descarga sea fácilmente rastreable.

    Este script facilita la integración y automatización del proceso de descarga de archivos C4C desde SharePoint, ideal para sistemas de gestión de archivos y procesamiento de datos.

    @log_step_decorator("Descargar archivo de SharePoint")
def download_file_from_sharepoint(file_name, folder_url, download_path):
    # Crear las carpetas si no existen
    if not os.path.exists(os.path.dirname(download_path)):
        os.makedirs(os.path.dirname(download_path))
        logging.info(f"Carpeta creada: {os.path.dirname(download_path)}")

    # Ruta completa del archivo en SharePoint
    logging.info(f"Ruta completa del archivo a descargar: {download_path}")

    download_url = f"{sharepoint_base_url}/_api/web/GetFolderByServerRelativeUrl('{folder_url}')/Files('{file_name}')/$value"
    response = requests.get(download_url, headers=headers)

    if response.status_code == 200:
        # Eliminar la marca de fecha y hora del nombre del archivo
        clean_file_name = file_name
        clean_download_path = os.path.join(os.path.dirname(download_path), clean_file_name)

        # Guardar el archivo descargado con el nuevo nombre
        with open(clean_download_path, "wb") as file:
            file.write(response.content)
        logging.info(f"Archivo descargado exitosamente como: {clean_file_name}")
    else:
        logging.info(f"Error al descargar el archivo. Código de estado: {response.status_code}")
        logging.info(f"Error: {response.text}")

    response = requests.get(folder_api_url, headers=headers)
