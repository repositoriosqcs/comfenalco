    01. ORIGEN Y MEDIDAS

    Origen de Datos del Cubo¶

    Descripción General¶

    El origen de datos es un componente estructurado que conecta el cubo de SSIS con el Data Warehouse principal. Esta conexión permite la extracción, transformación y carga (ETL) de datos necesarios para las operaciones analíticas.

    Detalles de Configuración¶

    1. Tipo de Origen:

    • Estructurado (structured).

    2. Nombre del Origen:

    • SQL/10 5 21 29\\bi;DWH_COMFENALCO.

    3. Detalles de Conexión:

    • Protocolo: tds (Tabular Data Stream, utilizado para bases de datos SQL Server).
    • Dirección:
      • Servidor: 10.5.21.29\bi.
      • Base de Datos: DWH_COMFENALCO.

    4. Autenticación:

    • Tipo: UsernamePassword.
    • Usuario: prov_quality2.
    • Conexión encriptada: false.

    5. Credenciales:

    • Ruta: 10.5.21.29\\bi;DWH_COMFENALCO.
    • Cifrado: No aplica (EncryptConnection: false).

    Propósito¶

    El origen de datos garantiza la conectividad y el acceso a las tablas y medidas necesarias para el funcionamiento del cubo. Este origen permite a los paquetes de SSIS consumir datos directamente desde el DWH para su análisis y explotación.

    Medidas¶

    Descripción General¶

    La tabla Medidas contiene columnas calculadas y una serie de medidas configuradas para realizar análisis y cálculos dentro del cubo. Estas medidas se utilizan para analizar datos clave y generar métricas significativas para la toma de decisiones.

    Estructura¶

    Cada medida calculada se describe brevemente, indicando su objetivo, la lógica DAX utilizada y (donde aplica) el formato de salida.

    1. # Unidades¶

      • Objetivo: Contar el número de unidades distintas en la tabla Transversal DIM_UNIDAD, excluyendo aquellas con ID igual a 5.

      • Expresión DAX:

      var dev = CALCULATE(
    DISTINCTCOUNT('Transversal DIM_UNIDAD'[ID_UNIDAD]), 
    'Transversal DIM_UNIDAD'[ID_UNIDAD] <> 5
)
return dev
      • Formato: Numérico.

    2. # Empresas Afiliadas Caja¶

      • Objetivo: Contar la cantidad de empresas afiliadas a la caja de compensación, considerando filtros de unidad según corresponda.

      • Expresión DAX:

      VAR EmpresasCalculo = 
    CALCULATE(
        DISTINCTCOUNT('Transversal FACT_ACTIVIDADES'[PARTNER]), 
        'Transversal FACT_ACTIVIDADES'[ID_EMPRESA] <> -1,
        'Transversal FACT_ACTIVIDADES'[ACTIVIDAD] = "AFILIACION"
    )

VAR EmpresasCalculoSinFiltros = 
    CALCULATE(
        DISTINCTCOUNT('Transversal FACT_ACTIVIDADES'[PARTNER]), 
        'Transversal FACT_ACTIVIDADES'[ID_EMPRESA] <> -1,
        'Transversal FACT_ACTIVIDADES'[ACTIVIDAD] = "AFILIACION",
        REMOVEFILTERS('Transversal DIM_UNIDAD'), 
        REMOVEFILTERS('Proteccion DIM_PROGRAMA')
    )

var dev = SWITCH (
    TRUE(),
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {1, 2, 3,4}, 
    EmpresasCalculoSinFiltros,
    EmpresasCalculo
)

RETURN IF( ISBLANK(dev) , 0 , dev)
      • Formato: Numérico.

    9. #Beneficiarios_Atendidos¶

      • Objetivo: Calcular el total de beneficiarios atendidos, considerando tanto beneficiarios formales como población no afiliada, y aplicando lógicas distintas según la unidad seleccionada.

      • Expresión DAX:

      VAR BeneficiariosAtendidos =
    CALCULATE(
        DISTINCTCOUNT('Transversal FACT_ACTIVIDADES'[PARTNER]), 
        'Transversal FACT_ACTIVIDADES'[ID_BENEFICIARIO] <> -1,
        'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> 5, 
        'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> BLANK()
    )

VAR BeneficiariosAtendidosColegio =
    CALCULATE(
        DISTINCTCOUNT('Transversal FACT_ACTIVIDADES'[PARTNER]), 
        'Transversal FACT_ACTIVIDADES'[ID_BENEFICIARIO] <> -1,
        'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> 5, 
        'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> BLANK(),
        'Transversal FACT_ACTIVIDADES'[ACTIVIDAD] = "ESTADO_MATRICULAS"

VAR PoblacionNoAfiliada =
    CALCULATE(
        DISTINCTCOUNT('Transversal FACT_ACTIVIDADES'[ID_POBLACION]), 
        'Transversal FACT_ACTIVIDADES'[ID_BENEFICIARIO] = -1,
        'Transversal FACT_ACTIVIDADES'[ID_AFILIADO] = -1,
        'Transversal FACT_ACTIVIDADES'[ID_EMPRESA] = -1,
        'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> 5, 
        'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> BLANK()
    )

VAR PoblacionNoAfiliadaColegio =
    CALCULATE(
        DISTINCTCOUNT('Transversal FACT_ACTIVIDADES'[PARTNER]), 
        'Transversal FACT_ACTIVIDADES'[ID_BENEFICIARIO] = -1,
        'Transversal FACT_ACTIVIDADES'[ID_AFILIADO] = -1,
        'Transversal FACT_ACTIVIDADES'[ID_EMPRESA] = -1,
        'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> 5, 
        'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> BLANK(),
        'Transversal FACT_ACTIVIDADES'[ACTIVIDAD] = "ESTADO_MATRICULAS"
    )

var dev = SWITCH (
    TRUE(),
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {1}, BeneficiariosAtendidosColegio + PoblacionNoAfiliadaColegio,
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {2}, BeneficiariosAtendidos + PoblacionNoAfiliada,
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {3}, 0,
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {4}, BeneficiariosAtendidos + PoblacionNoAfiliada,
    BeneficiariosAtendidos
)

RETURN IF( ISBLANK(dev) , 0 , dev)
      • Formato: Numérico.

    106. # Retiros¶

      • Objetivo: Calcular el número total de retiros registrados, teniendo en cuenta las diferentes unidades y periodos.

      • Expresión DAX:

      VAR colegio_retiros = CALCULATE(
    DISTINCTCOUNTNOBLANK('Transversal FACT_ACTIVIDADES'[PARTNER]),
    'Transversal FACT_ACTIVIDADES'[ACTIVIDAD] = "RETIROS",
    'Transversal FACT_ACTIVIDADES'[ID_BENEFICIARIO] <> -1,
    'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> 5, 
    'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> BLANK(),
    DATESYTD('Transversal DIM_TIEMPO_MENSUAL'[FECHA])
)

VAR colegio_retirosNoAfiliada = CALCULATE(
    DISTINCTCOUNTNOBLANK('Transversal FACT_ACTIVIDADES'[PARTNER]),
    'Transversal FACT_ACTIVIDADES'[ACTIVIDAD] = "RETIROS",
    'Transversal FACT_ACTIVIDADES'[ID_BENEFICIARIO] = -1,
    'Transversal FACT_ACTIVIDADES'[ID_AFILIADO] = -1,
    'Transversal FACT_ACTIVIDADES'[ID_EMPRESA] = -1,
    'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> 5,
    'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] <> BLANK(),
    DATESYTD('Transversal DIM_TIEMPO_MENSUAL'[FECHA])
)

VAR InicioSemestre = IF(
    MONTH(MINX(DATESYTD('Transversal DIM_TIEMPO_MENSUAL'[FECHA]), 'Transversal DIM_TIEMPO_MENSUAL'[FECHA])) <= 6,
    DATE(YEAR(MINX(DATESYTD('Transversal DIM_TIEMPO_MENSUAL'[FECHA]), 'Transversal DIM_TIEMPO_MENSUAL'[FECHA])), 1, 1), -- Primer semestre
    DATE(YEAR(MINX(DATESYTD('Transversal DIM_TIEMPO_MENSUAL'[FECHA]), 'Transversal DIM_TIEMPO_MENSUAL'[FECHA])), 7, 1) -- Segundo semestre
)

VAR FinSemestre = IF(
    MONTH(MAXX(DATESYTD('Transversal DIM_TIEMPO_MENSUAL'[FECHA]), 'Transversal DIM_TIEMPO_MENSUAL'[FECHA])) <= 6,
    DATE(YEAR(MAXX(DATESYTD('Transversal DIM_TIEMPO_MENSUAL'[FECHA]), 'Transversal DIM_TIEMPO_MENSUAL'[FECHA])), 6, 30), -- Fin del primer semestre
    DATE(YEAR(MAXX(DATESYTD('Transversal DIM_TIEMPO_MENSUAL'[FECHA]), 'Transversal DIM_TIEMPO_MENSUAL'[FECHA])), 12, 31) -- Fin del segundo semestre
)

VAR RetirosCedesarrollo = CALCULATE(
    DISTINCTCOUNT('Transversal FACT_ACTIVIDADES'[ID_DESERCION]),
    'Transversal FACT_ACTIVIDADES'[ACTIVIDAD] = "RETIROS",
    'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] = 2,
    DATESBETWEEN('Transversal DIM_TIEMPO_MENSUAL'[FECHA], InicioSemestre, FinSemestre),
    REMOVEFILTERS('Transversal FACT_ACTIVIDADES'[ID_UNIDAD])
)

VAR RetirosDE = CALCULATE(
    DISTINCTCOUNT('Transversal FACT_ACTIVIDADES'[ID_DESERCION]),
    'Transversal FACT_ACTIVIDADES'[ACTIVIDAD] = "RETIROS",
    'Transversal FACT_ACTIVIDADES'[ID_UNIDAD] = 3,
    REMOVEFILTERS('Transversal FACT_ACTIVIDADES'[ID_UNIDAD])
)

VAR dev = SWITCH(
    TRUE(),
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {1}, colegio_retiros + colegio_retirosNoAfiliada,
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {2}, RetirosCedesarrollo,
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {3}, RetirosDE,
    SELECTEDVALUE('Transversal DIM_UNIDAD'[ID_UNIDAD]) IN {4}, 0,
    colegio_retiros + colegio_retirosNoAfiliada + RetirosCedesarrollo + RetirosDE
)

RETURN IF(ISBLANK(dev), 0, dev)
      • Formato: Número.
